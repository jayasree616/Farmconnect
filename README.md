🚀 About the Project

FarmConnect is a lightweight, web-based platform built to connect Farmers who need help with agricultural work and Laborers who are looking for jobs.

It provides a clean UI, easy job posting, instant applications, and smooth communication — all running on HTML, CSS, JavaScript, and a JWT-secured Node.js backend.

This project was created as part of a real-world learning journey and showcases full-stack development, deployment, and role-based authentication.

✨ Key Features
👨‍🌾 Farmer Features

✔ Post agricultural jobs
✔ Edit or delete postings
✔ View number of applicants
✔ View applicant contact numbers
✔ Manage jobs from dashboard

👷 Laborer Features

✔ View all available jobs
✔ Filter jobs by Location / Work Type
✔ Apply for jobs instantly
✔ See job details clearly

🔐 Authentication

✔ Signup/Login with JWT
✔ Secure token storage
✔ Role-based UI (Farmer/Admin vs Laborer/User)
